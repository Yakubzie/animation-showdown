How to Use Performance Testing

  1. Performance monitoring is available on GSAP and Vanilla JS demo pages
  2. Look for the performance panel (─ icon) in the bottom-right corner
  3. Click the ─ button to collapse/expand the panel for better mobile experience
  4. Use "Start" to begin monitoring performance during animations
  5. Navigate between GSAP and Vanilla JS demos to compare performance
  6. Click "Run Tests" to execute automated performance tests
  7. Note: Tests run slower on demo pages due to heavy animations
  8. View detailed analysis and recommendations in the Analysis section

Key differences between GSAP and Vanilla JS approaches at a glance

G

GSAP

Advantages

  • Timeline control for complex sequences
  • Rich easing functions library
  • ScrollTrigger plugin
  • Excellent browser compatibility

Considerations

  • ~80kb bundle size
  • Commercial licensing
  • Learning curve

V

Vanilla JS

Advantages

  • Zero dependencies
  • Smaller bundle size
  • Native browser APIs
  • Full control over implementation

Considerations

  • More code required
  • Limited to modern browsers
  • Manual implementation
